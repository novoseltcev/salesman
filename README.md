# salesman
 solving the traveling salesman problem using the branch and boundary method

***example 1***
```python
    |   1       2       3       4       5  
----|-------------------------------------
  1 |  0.0	20	18	12	8
  2 |  5	0	14	7	11
  3 |  12	18	0	6	11
  4 |  11	17	11	0	12
  5 |  5	5	5	5	0
Weight = 41.0
Way = '2 -> 1 -> 5 -> 3 -> 4 -> 2'
```
